# Fast water and H3 for Alanine Dipeptide

This is an example of alanine dipeptide. In total this system has 2449 atoms. There are two examples of mass update files (H3.dat and fast\_water.dat) that have a record for each value. In accordance with the massUpdates functionality, values of 0 indicate that the mass in the PSF file is used. Any values > 0 will set the mass to this value. The NAMD configuration file template.namd shows how these mass update files are indicated in the configuration.
